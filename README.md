GPS Flutter Codex

Aplicación desarrollada en Flutter que obtiene la ubicación GPS del dispositivo utilizando la librería **Geolocator**. Este proyecto fue realizado como parte de la práctica de comparación entre Flutter e Ionic, empleando herramientas de inteligencia artificial para el desarrollo asistido.

Objetivo

Implementar una aplicación móvil capaz de obtener la ubicación actual del usuario mediante el GPS del dispositivo, mostrando la latitud y longitud en pantalla.

Características

- Obtención de la ubicación actual.
- Solicitud automática de permisos.
- Visualización de la latitud.
- Visualización de la longitud.
- Manejo de errores cuando el GPS está desactivado o los permisos son denegados.

## Tecnologías utilizadas

- Flutter
- Dart
- Geolocator

## Dependencias

```yaml
geolocator
```

## Instalación

1. Clonar el repositorio.

```bash
git clone <URL_DEL_REPOSITORIO>
```

2. Entrar al proyecto.

```bash
cd gps_flutter_codex
```

3. Instalar dependencias.

```bash
flutter pub get
```

4. Ejecutar la aplicación.

```bash
flutter run
```

## Funcionamiento

1. Abrir la aplicación.
2. Presionar el botón **Obtener Ubicación**.
3. Conceder permisos de ubicación cuando el sistema lo solicite.
4. La aplicación mostrará la latitud y longitud actuales del dispositivo.
